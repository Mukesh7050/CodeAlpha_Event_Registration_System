# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Mukesh7050/CodeAlpha_Event_Registration_System.git
```

---

## Backend Setup

```bash
cd Backend

npm install
```

---

## Create .env

```
JWT_SECRET=your_secret_key

MONGO_URI=your_mongodb_connection
```

---

## Start Backend

```bash
npm start
```

---

## Frontend

Open

```
Frontend/index.html
```

using Live Server.

---

# 📌 API Endpoints

## User

```
POST /api/users/signup

POST /api/users/login
```

---

## Events

```
GET /api/events

POST /api/events

GET /api/events/:id
```

---

## Registration

```
POST /api/registrations

GET /api/registrations/my

DELETE /api/registrations/:id
```
